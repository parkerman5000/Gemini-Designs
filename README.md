# Gemini Designs

**Merchandise Creator Division of mpowerio.ai LLC**

Brand aesthetic: *Farmhouse Chic meets Solarpunk*

## About

Led by Project Manager Maria Rojas, Gemini Designs brings artisan quality and personal attention to every merchandise project. We help businesses make a lasting impression through custom-crafted wearables and promotional items.

## Services

- **Embroidery** - Professional embroidered logos on polos, hats, jackets, bags
- **Heat Transfer (HTF)** - Vibrant vinyl designs for apparel and accessories
- **Custom Stickers** - Precision-cut stickers and decals using Cricut
- **Vehicle Magnets** - Weatherproof custom magnets for cars and fleet vehicles
- **DTF T-Shirts** - Direct-to-Film printing for instant custom merch *(Coming Soon)*

```bash
# Watch for changes during development
npm run dev

# Build for production
npm run build
```
